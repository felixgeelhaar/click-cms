
## Core Plugin System

Full WordPress-like plugin architecture with hooks, events, lifecycle (install/activate/deactivate/uninstall), routing, and dependency resolution. Core of the CMS extensibility.

---

## Storage Abstraction

Pluggable storage backends with unified interface. Default JSON files (Bludit-style), with plugins for SQLite, MySQL, PostgreSQL support.

---

## API-First Headless

REST API for headless content access. Default API endpoints for pages, content, users. Plugins can extend with custom endpoints.

---

## Admin UI Plugin

Optional admin panel plugin for non-engineers to manage content, plugins, and settings. Easy deployment via admin UI.

---

## Plugin Marketplace

Easy plugin discovery and installation. ZIP upload, marketplace browse, auto-updates for plugins.
